🎓 STUDENTS MARKLIST MANAGEMENT SYSTEM

A web-based Students Marklist Management System developed using:

  🎨 Frontend: HTML, CSS
  
  ⚙ Backend: Python (Django)
  
  🗄 Database: MySQL

  

This project helps manage student details and their academic marks efficiently.


📌 Project Description

The Students Marklist Management System is designed to:

  * Register students
  * Store student details
  * Add and manage student marks
  * View mark lists
  * Perform CRUD operations (Create, Read, Update, Delete)
  * Provide secure login functionality

This system is suitable for schools or institutions to digitally manage student records.




🚀 Features

✅ Student Registration

✅ Login Authentication

✅ Add Student Marks

✅ View Student Marklist

✅ Update Marks

✅ Delete Records

✅ Admin Panel (Django Admin)

✅ MySQL Database Integration
